# Digital-Signal-Processing-Demo
本项目记录了《数字信号处理》课程中常见的仿真题目，完全采用python环境，摆脱了该课程对Matlab软件的依赖

由于一些众所周知的原因，matlab难以纯净下载且软件本体动辄几十G，但广大高校的《数字信号处理》和《信号与系统》课程中有带有仿真作业，给我们带来了极大的困扰。

本项目旨在为这类教学仿真提供一个便捷，轻量化，好用的解决思路，收录了一些常见的作业。
## 使用方法
* 先安装python解释器，具体方法咨询各大LLM大语言模型
* 建立虚拟环境
```cmd
python -m venv .venv
```
* 激活虚拟环境
  * VScode中直接在最上面输入`>Python:Select Interpriter`并选择.venv
* 在虚拟环境中执行
```cmd
pip install -r requirements.txt
```
* 然后就可以正常运行所有的.py代码了
